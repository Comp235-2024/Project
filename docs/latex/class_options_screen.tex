\doxysubsection{Detailed Description}
Represents the options screen state of the game. 

The \doxylink{class_options_screen}{Options\+Screen} class is derived from the \doxylink{class_state}{State} class and represents the state of the game when the options screen is active. It provides methods to initialize, handle input, update, and draw the options screen.
