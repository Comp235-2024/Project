\doxysubsection{Detailed Description}
Represents a campaign consisting of multiple maps and their connections. 

The \doxylink{class_campaign}{Campaign} class provides functionality to manage a campaign, including adding and removing maps, creating connections between maps, editing maps, and saving/loading the campaign to/from a JSON file.
