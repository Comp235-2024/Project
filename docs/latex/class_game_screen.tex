\doxysubsection{Detailed Description}
Represents the game screen state. 

The \doxylink{class_game_screen}{Game\+Screen} class is derived from the \doxylink{class_state}{State} class and represents the state of the game screen. It handles the initialization, input handling, updating, and drawing of the game screen.
