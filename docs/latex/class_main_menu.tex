\doxysubsection{Detailed Description}
Represents the main menu state of the game. 

The \doxylink{class_main_menu}{Main\+Menu} class inherits from the \doxylink{class_state}{State} class and provides functionality for initializing, handling input, updating, and drawing the main menu state of the game.
