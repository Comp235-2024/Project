\doxysubsection{Detailed Description}
Represents the welcome screen state of the game. 

The \doxylink{class_welcome_screen}{Welcome\+Screen} class is derived from the \doxylink{class_state}{State} class and is responsible for managing the welcome screen state of the game. It handles initialization, input handling, updating, and drawing of the welcome screen.
