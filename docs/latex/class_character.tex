\doxysubsection{Detailed Description}
Represents a character in a game. 

The \doxylink{class_character}{Character} class stores information about a character\textquotesingle{}s ability scores, hit points, armor class, attack bonus, damage bonus, and level. It also provides methods to generate ability scores and calculate ability modifiers, hit points, armor class, attack bonus, and damage bonus.
