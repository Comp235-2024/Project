\doxysubsection{Detailed Description}
Represents a pillar object in the game. 

The \doxylink{class_pillar}{Pillar} class is a derived class of the \doxylink{class_movable}{Movable} class. It represents a pillar object that can be moved in the game. It inherits the properties and methods of the \doxylink{class_movable}{Movable} class and adds a texture\+Name member variable.
