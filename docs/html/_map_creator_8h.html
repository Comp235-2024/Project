Detailed Description

This class is called from the main menu and allows the user to create a map by placing objects on a grid and then either save or clear that map.

Author
Ricardo Raji Chahine 40234410

The user can place walls, doors, levers, chests, players, and npcs on the map.

The maps are saved as .json files in the Saved Maps directory.
