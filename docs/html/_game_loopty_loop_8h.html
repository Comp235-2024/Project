Header file for the GameLooptyLoop class.

This file defines the GameLooptyLoop class, which represents the main game loop of the application. The GameLooptyLoop class is responsible for managing the game state, handling user input, and updating the game logic. It also contains a MainData struct that holds various game-related data, such as the state machine, window, asset manager, input manager, log observer, and campaign.
